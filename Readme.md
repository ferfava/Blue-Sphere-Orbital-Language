# Blue Sphere: Orbital Language Edition 🛰️

**Applying NLP to satellite telemetry: treating orbital data as a linguistic corpus for semantic sonification.**

## 🚀 Descripción del Proyecto
Blue Sphere explora la telemetría satelital (datos de sensores) tratándola como si fuera un lenguaje con su propia gramática y semántica. 

## 🛠️ Tecnologías
- **PLN:** spaCy / NLTK (para tokenización de datos).
- **Data:** SatNOGS API (telemetría real).
- **Audio:** Librosa / gTTS (sonificación).

## 📂 Estructura
- `src/`: Scripts de procesamiento.
- `data/`: Datos crudos del espacio.
